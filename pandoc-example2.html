Invoking Pandoc

To Make a PDF

Use the following to create a PDF from a markdown file.

c:\> pandoc eg-pandoc.md -o eg-pandoc.pdf

To Make HTML5

c:\> pandoc -s -c pandocbd.css -t html5 -o pandoc-example2.html pandoc-example2.md
